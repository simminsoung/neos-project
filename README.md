# NEOS
- 대학생 스터디 모집 커뮤니티 플랫폼


# 프로젝트 목적
대학생들을 위한 스터디 모집 
관심 분야의 전공 등 서로의 공부법을 공유 
대학생뿐만 아니라 일반 중고등학생도 평소에 관심있는 학과에
대해 정보를 받을수 있으며 자료를 사고파는 자료상점도 이용가능
일반 커뮤니티와 자신의 고민상담을 올릴수있는 익명 고민상담 서비스를
운영 하고있으며 서로의 부족했던 부분을 채워줄수 있는 
웹 서비스 입니다.
